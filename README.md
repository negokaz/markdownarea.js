# markdownarea.js

In-browser, markdown documentation tool.

## Usage

```html
<!DOCTYPE html>
<html>
<body>
<textarea class="markdownarea">
# Markdown here!
</textarea>
<script src="https://cdn.jsdelivr.net/gh/negokaz/markdownarea.js/dist/markdownarea.js"></script>
</body>
</html>
```

The html file is rendered by parsed markdown.
