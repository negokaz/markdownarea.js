# markdownarea.js

## Usage

```html
<!DOCTYPE html>
<html>
<body>
<textarea class="markdownarea">
# Markdown here!
</textarea>
<script src="https://cdn.jsdelivr.net/gh/negokaz/markdownarea.js/dist/markdownarea.js"></script>
</body>
</html>
```
